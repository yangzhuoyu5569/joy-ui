# joy-vue

## 安装依赖
```
npm install
```

## 通过如下操作解决 npm 下载速度慢的问题
```
npm install --registry=https://registry.npmmirror.com
```

### 运行
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
